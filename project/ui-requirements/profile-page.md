# Profile Page — UI Requirements

---

## Access
Via Hamburger menu → Profile link.

## Top Navigation
- Left: Back arrow
- Center: "Profile" — Playfair, 24px
- Right: Edit icon (pencil) → toggle edit mode

## Profile Header (Centered)
```
Profile photo:
  Shape:          Circle, 80px diameter
  Default:        Icon placeholder (person icon, #3A3A3A bg)
  Tap to change:  Open camera/gallery
  Border:         2px #DB8686 (accent)

Name:
  Font:           Playfair, 28px, white
  Position:       Below photo, centered

Age & Gender:
  Format:         "24 • Male"
  Font:           Plex Mono, 14px, #A0A0A0
  Center aligned
```

## Promise Section
```
Header section label:  "Your Promise" — Playfair, 20px
Promise text:
  Font:           Playfair, 24px, #DB8686
  Style:          Italic, quoted
  Centered
  Example:        "Build discipline to achieve my goals"
Edit icon:        Small pencil icon, top-right of section
  Action:         Opens modal with text input (10-word limit)
```

## Quick Stats Section
```
Header:  "Quick Stats" — Playfair, 20px
Grid (2 columns):
  "45 days tracked"
  "78% overall"
  "23 days streak (No-Fap)"
  "342 total entries"
Font: Plex Mono, 16px  
Label: Plex Mono, 12px, #A0A0A0 (below number)
```

## Privacy & Legal Section
```
Header:  "Privacy & Legal" — Playfair, 20px

List items (tappable rows):
  Privacy Policy  → Opens modal or external link
  Terms & Conditions → Opens modal or external link
  Data Management → Navigate to Settings/Data section

Each row:
  Left:  Text (Plex Mono, 16px, white)
  Right: Chevron icon (#A0A0A0)
  Height: 52px
  Border bottom: 1px #2A2A2A
```
