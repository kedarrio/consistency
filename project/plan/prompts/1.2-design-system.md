## Task

Implement the complete design system for the Consistency app.

## Context

All design values are in `project/design/tokens.json`. All rules in `project/design/claude-guardrails.md`.

## Steps

1. Create `src/styles/tokens.css` with CSS custom properties for all token values

2. Create `src/styles/globals.css`:
   - Import tokens.css
   - Set body background to `var(--color-bg-primary)`, color to `var(--color-text-primary)`
   - Import Google Fonts (Playfair Display 600, IBM Plex Mono 400)
   - Set heading font to Playfair Display globally
   - Remove all default margins/padding (CSS reset)

3. Update `tailwind.config.js` to reference custom properties

4. Test by creating a `DesignSystemTest` page that shows:
   - All colors with their tokens
   - Both fonts at each size
   - Spacing scale visualization
