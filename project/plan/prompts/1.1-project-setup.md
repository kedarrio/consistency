# Prompt 1.1 — Project Setup

> Copy and paste this prompt to an AI agent to complete Phase 1.1.

---

## Task

Set up a new Vite + React project for the Consistency habit tracking PWA.

## Context

- App name: **Consistency**
- Tech stack: React 18+, Vite, React Router v6, Tailwind CSS, Material Design Icons, Recharts, date-fns
- Design tokens are in `project/design/tokens.json`
- Agent rules are in `project/design/claude-guardrails.md`

## Steps

1. Initialize Vite + React in the current directory:
   ```bash
   npx create-vite@latest . --template react
   ```

2. Install all dependencies:
   ```bash
   npm install react-router-dom recharts date-fns
   npm install @mui/icons-material @mui/material @emotion/react @emotion/styled
   npm install framer-motion
   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   ```

3. Configure Tailwind (`tailwind.config.js`) with custom theme extending the tokens from `project/design/tokens.json`:
   - Add all colors as Tailwind theme colors
   - Add font families (Playfair Display, IBM Plex Mono)
   - Add spacing scale

4. In `index.html`, add Google Fonts link for Playfair Display and IBM Plex Mono

5. Create folder structure:
   ```
   src/
   ├── components/
   │   ├── buttons/
   │   ├── cards/
   │   ├── inputs/
   │   ├── navigation/
   │   ├── progress/
   │   └── charts/
   ├── pages/
   ├── context/
   ├── utils/
   ├── hooks/
   └── assets/
   ```

6. Set up React Router with stub routes for: `/`, `/stats`, `/calendar`, `/settings`, `/profile`, `/habit/:id`, `/logs/:id`, `/onboarding`

7. Run `npm run dev` to verify everything compiles without errors

## Acceptance Criteria

- `npm run dev` runs without errors
- Fonts load (verify in browser)
- Routing works between stub pages
- File structure matches spec above

## Reference Files

- `project/design/tokens.json` — all design values
- `project/design/claude-guardrails.md` — rules to follow
